موقع تعليمي بسيط لتعليم اللغة الإنجليزية مع معلم ذكي AI

المتطلبات:
- Node.js 16+ (تحميل من https://nodejs.org)
- مفتاح Anthropic API (انسخ من https://console.anthropic.com)

التثبيت والتشغيل:
1. فك ضغط/انسخ المشروع
2. افتح Terminal في المجلد وشغّل:
   npm install
3. أنشئ ملف `.env` وأضف مفتاحك:
   ANTHROPIC_API_KEY=your_actual_key_here
   PORT=3000
4. شغّل الخادم:
   npm start
5. افتح http://localhost:3000 في المتصفح

الميزات:
- صفحة رئيسية تعليمية بالعربية
- دروس بسيطة قابلة للتوسيع
- معلم AI ذكي (Claude) للإجابة على أسئلة اللغة الإنجليزية
- واجهة دردشة تفاعلية مع الحفظ التاريخ

ملاحظات أمان:
- تخزين المفاتيح في .env (لا تشارك هذا الملف)
- API يعمل من الخادم فقط (أمان أفضل)

تطويرات مستقبلية:
- تسجيل المستخدمين وحفظ التقدم
- ملفات صوتية للنطق
- تمارين تقييمية
- واجهة ملكة للمعلمين
